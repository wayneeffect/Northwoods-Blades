## About the Project

**Northwoods Blades** is a professional demo website for an online store specializing in:

- Pocket knives & tactical knives  
- Kitchen cutlery  
- Swords & collectibles  
- Rifle scopes & optics  

Designed with a rugged, outdoorsy feel that reflects the **Upper Peninsula of Michigan** lifestyle (Newberry, Manistique, Naubinway area).

### Features

- Fully responsive design (mobile-first)
- Modern Tailwind CSS styling
- Hero section with strong branding
- Category showcase
- Featured products grid
- Smooth scrolling navigation
- Clean, professional footer
- Easy to customize

### Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Font Awesome** icons
- Vanilla JavaScript (minimal)
- No build tools required

### Deployment (Recommended)

This site is built to be deployed as a **static website**:

1. Fork or clone this repository
2. Push to GitHub
3. Connect your repo to **[Render.com](https://render.com)** → New Static Site
4. Deploy in under 2 minutes (no build command needed)

### Customization

- Edit text, prices, and products directly in `index.html`
- Replace placeholder images (using `picsum.photos`) with real product photos
- Change colors in the `<style>` tag or by editing Tailwind classes
- Add more products or pages as needed

### Future Plans (Optional)

- Convert to full Shopify or WooCommerce store
- Add real shopping cart (Snipcart or similar)
- Blog section
- Product filtering
